사용 방법

공간 설정

  • 좌측 공간 크기에서 가로/세로(cm)를 입력하고 공간 적용을 누르면 방 크기가 바뀝니다.
  • 우측 상단의 확대 슬라이더로 화면 배율을 조절할 수 있습니다.

책상 · 기둥 · 문 배치

  • 좌측 패널의 크기 입력 후 추가(책상), 기둥 추가, 문 추가 버튼으로 항목을 만들 수 있습니다.
  • 공간 안의 항목을 드래그하면 위치를 옮길 수 있습니다. 10cm 단위로 스냅됩니다.
  • 항목을 클릭해 선택한 뒤 좌측의 X / Y / 가로 / 세로 입력으로 정확한 값을 지정할 수 있습니다.
  • 선택 항목은 90도 회전, 복제, 삭제 버튼으로 다룰 수 있습니다.

이름 입력

  • 책상/기둥/문 위의 이름 라벨을 클릭하면 그 자리에서 바로 이름을 입력할 수 있습니다.
  • Enter 또는 바깥쪽 클릭으로 저장, Esc로 취소합니다.

저장 / 불러오기

  • 모든 변경 사항은 자동으로 브라우저에 저장되며, 다시 열어도 마지막 상태가 유지됩니다.
  • 우측 상단 Export는 현재 배치를 JSON 파일로 내려받습니다.
  • Import는 JSON 파일을 불러와 현재 배치를 덮어씁니다.

키보드 단축키

  • Delete — 선택한 항목 삭제
  • R — 선택한 항목 90도 회전
